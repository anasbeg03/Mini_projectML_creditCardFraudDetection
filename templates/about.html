About FraudShield

This web application detects fraudulent credit card transactions using machine learning.

Project Details:
  • Dataset: Kaggle - Credit Card Fraud Detection
  • ML Models: Logistic Regression, Random Forest, XGBoost, Support Vector Machine
  • Features Used: V10, V12, V14, V16, V17, Amount
  • Tech Stack: Python, Flask, Bootstrap, scikit-learn, XGBoost

This project was built for educational purposes to demonstrate fraud detection using real-world data.
